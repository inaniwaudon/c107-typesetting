# c107-typesetting

コミックマーケット 107 にて頒布した同人誌「組版処理 Vol.1」に関するリポジトリです。

以下の記事に対応するソースコードを含んでいます。

- `two-columns`：「二段組の組版技術」
- `latex`：「LaTeX で「どの単語・行を削ればページ内に収まるか」を可視化するスクリプト」
